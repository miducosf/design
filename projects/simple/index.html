Mentor Chat — Scenario-Based Coaching Simulator

Scenario practice with immediate coaching feedback, accessible media, and realistic stakeholder dialogue.

  • Role: Learning Experience Designer
  • Focus: Scenario-based practice, Coaching, Accessibility
  • Tools: HTML/CSS/JS, Synthesia (VTT), Render
  • Duration: 2 weeks

Overview

Mentor Chat is a lightweight practice environment for new managers. Learners explore realistic channels (email, chat, video) and receive immediate coaching feedback. The experience emphasizes accessible video, including closed captions via WebVTT, clear keyboard navigation, and high-contrast UI.

Process & Highlights

Design

  • Mapped a short scenario arc with clear decision points and feedback criteria.
  • Kept response options concise with distinct tone and intent.
  • Wrote coaching text that reinforces specific behaviors.

Build

  • Modular HTML & CSS; unobtrusive JS for state and layout.
  • Embedded video with <track kind="captions"> for CC via VTT.
  • Responsive layout: side-by-side on wide, stacks on narrow.

Iterate

  • Refined copy for clarity and coaching tone.
  • Simplified layout to avoid “mystery meat” interactions.
  • Added debug toggles to speed layout adjustments.

What I’d do next

  • Add a transcript auto-render under videos for SEO + a11y.
  • Track attempt analytics to personalize practice.
  • Swap in enterprise player with analytics + chapters.

Accessibility

  • ✓ Closed captions via WebVTT (kind="captions") on all videos.
  • ✓ Logical heading structure & labeled controls.
  • ✓ Keyboard navigable (visible focus) and reduced motion friendly.
  • ✓ Color contrast checks for copy and UI elements.
