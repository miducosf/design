Project - ACE AI Coach

The Challenge

Getting content approved by Compliance should be straightforward—but often, it’s not. Complex procedures, layered documentation requirements, and confusion around roles and timing can overwhelm even experienced professionals.

  • A one-size-fits-all course can help, but risks feeling irrelevant for veterans and frustrating for newcomers seeking quick answers.
  • Reference guides can help—if you know where to look.
  • Support teams can help—but often end up answering the same questions that a smarter solution could handle more efficiently.

The Insight

Instead of developing yet another static course, I asked:
“What if the learning experience met users in their moment of need?”
With recent advances in AI, I realized I could prototype a conversational coach that adapts to users' level of knowledge and specific questions.

The Solution

ACE (Assistant for Compliance Excellence) is an AI-powered digital coach that delivers efficient, contextual guidance—without forcing users to wade through irrelevant content.
ACE: 

  • Delivers just-in-time support based on real user queries.
  • Reduces reliance on long‑format courseware.
  • Improves efficiency and engagement by skipping what's not needed.
  • Creates a scalable framework for future AI-driven learning tools.
View Live Chatbot View Demo

The Reflection

  • ACE shows how learning designers can use AI not just to generate content—but to build adaptive support systems that grow with the learner.
  • This project also sharpened my full-stack prototyping skills while staying rooted in what matters most: solving real problems for real users.
  • The next iteration? A fully Retrieval-Augmented Generation (RAG) model that allows ACE to reference multiple uploaded documents—making answers smarter, faster, and more grounded in real process logic.

The Buildout

  • Designed a modern, bubble-style UI that feels familiar and engaging.
  • Developed an relatable AI persona that builds trust and delivers insights.
  • Connected the frontend to OpenAI's GPT API using a secure Node.js backend.
  • Styled the interface for clarity, accessibility, and professionalism.

The Timeline

Day 1: Ideation, layout wireframing, and conversation flow design

Day 2: Frontend development, styling, and mascot integration

Day 3: Backend integration, testing, refinement, and live deployment
