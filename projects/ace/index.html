Project - ACE AI Coach

The Challenge

Getting content approved by Compliance should be straightforward—but often, it’s not. Complex procedures, layered documentation requirements, and unclear roles can overwhelm even experienced professionals. Traditional learning formats fall short of offering immediate, usable answers.

Considerations

  • Experienced professionals often need quick reminders—not a course.
  • Newcomers need clarity and real-time guidance.
  • Reference guides can be helpful—if you know where to look.
  • Support teams end up fielding the same repetitive questions.

The Solution

ACE (Assistant for Compliance Excellence) is an AI-powered digital coach that delivers just-in-time, contextual support—without requiring users to sift through irrelevant content.

  • Responds to real user queries with focused, helpful answers
  • Reduces reliance on long-format courseware
  • Improves user efficiency and engagement
  • Creates a scalable foundation for future AI-powered tools
View Live Chatbot

Reflection

  • ACE shows how learning designers can use AI not just to generate content—but to build adaptive support systems that grow with the learner.
  • This project also sharpened my full-stack prototyping skills while staying rooted in what matters most: solving real problems for real users.
  • The next iteration? A fully Retrieval-Augmented Generation (RAG) model that allows ACE to reference multiple uploaded documents—making answers smarter, faster, and more grounded in real process logic.

Buildout

  • Designed a modern, bubble-style UI that feels familiar and engaging.
  • Developed a relatable AI persona that builds trust and delivers insights.
  • Connected the frontend to OpenAI's GPT API using a secure Node.js backend.
  • Styled the interface for clarity, accessibility, and professionalism.

Timeline

Day 1: Ideation, layout wireframing, and conversation flow design

Day 2: Frontend development, styling, and mascot integration

Day 3: Backend integration, testing, refinement, and live deployment
